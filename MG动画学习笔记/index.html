MG动画学习笔记

发表于 2019-04-23 | 更新于 2019-06-07 | 分类于 学习 | 评论数：

ctrl+shift+h 隐藏/显示变换控件

  • 创建形状图层时，使用矩形工具创建描边和填充，可以选择描边和填充的层级
    扩大描边半径时，描边会同时向内和向外扩大；如果将填充至于描边之上，那么填充将盖住描边，此时再扩大描边半径，则只会向外扩大
    • line cap(线段连接)：决定线段其起始和结尾的类型
    • line join（线段连接）：决定线段拐角处的链接方式
    • miter linmit（尖角限制）：通过调整数值来决定拐角处是否出现尖角；按ctrl进行拖动是以0.1为单位
    • dashes(虚线)：将描边转换为虚线效果，需要点击+号才能添加属性

  • 填充和描边渲染的顺序是由内而外的，如果形状本身有填充或描边，那么添加在它上一层级上的填充将没有效果，添加描边的话必须要超过形状原始描边的半径才会显现出来

  • 点击contents（内容）后面的add（添加）可以创建组，可以将同类元素放到一个组里，这样添加描边或填充时会方便一点

合并路径
  • 对形状图层里面的子图层，可以点击内容后面add添加merge path(合并路径)（必须是内容后面的add），然后会新生成填充和描边，强制统一填充和描边
  • merge path(合并路径)必须放在两个形状图层之后，放在其他地方都是不能生效的
offset path(位移路径)
  • 根据数值将当前形状图层进行扩大或缩小（变换控件/锚点位置不变）
repeater(中继器)
  • 类似C4D中的克隆
  • 添加中继器时不能直接点击contents（内容）后面的add（添加），这样创建的中继器是和子形状图层是同级的，对中继器进行位置、旋转、缩放时会以上一层级的形状图层的锚点位置作为中心；
  • 必须是选中需要添加中继器的图层然后再点击add进行添加，此时创建的中继器是该图层的子物体，这样的位置坐标才是正确的
  • 在使用中继器时最好使用矩形工具创建形状图层，这样才方便移动矩形路径自身的锚点和中继器旋转中心
  • 当形状路径使用中继器之后不能修改路径形状，否则就会造成异常显示，一句话来说就是如果要使用中继器，那创建形状图层必须一步到位

  • 通过变换：中继器下的锚点来控制中继器对象的旋转中心，默认是（0，0）

  • 圆角：添加后对形状图层的顶点进行圆滑处理

  • 修剪路径：对路径的开始和结束进行K帧动画
  • 抖动路径：类似于网易云音乐圆圈效果，就像是给形状图层添加了一个动态噪波效果，可以根据大小数值调节幅度
    • 此路径操作的多个属性的行为与用于文本动画的摆动选择器的同名属性的行为相同
    • “关联”属性指定顶点的运动与其邻点的运动之间的相似程度；值越小，锯齿效果越明显，因为顶点的位置对其邻点位置的依赖程度更小。

-摆动变换

    • 随机分布（摆动）路径的位置、锚点、缩放和旋转变换的任意组合。
    • 摆动变换”操作在中继器操作之后尤其有用，因为它允许您单独随机分布每个重复的形状的变换

  • Z字形：使形状图层的路径生成像折线锯子一样的折现

  • 通过圆角矩形创建形状时，可以通过鼠标滚轮进行圆角半径调整

  • AE中的形状图层本身不像PS中的一样可以直接变形，如果要实现变形效果，则需要点击contents>rectangle>rectangle path 1，然后右键转换为贝塞尔曲线
填充与描边
  • 形状图层本身可以设置填充与描边

  • 一级形状图层不设置描边与填充，那么内容里面的其他形状图层也不能设置填充与描边

  • 修改线段圆角时首先尝试线段端点，如果线段端点没有用的话那就尝试线段链接

  • 渐变填充会腐蚀部分描边，从而使描边半径减小
  • 如果使用钢笔工具创建路径，那么就会生成一个新的形状图层；如果使用多边形工具进行创建，那么可以在选中的层级下生成，而不会生成新的图层。

  • 使用脚本创建动画时需要关键帧为默认关键帧，即为匀速关键帧

  • 如果将关键帧类型转换为缓动，那么表达式创建的效果则不能生效，此时则需要打开图标编辑器，并且打开表达式图层，调整出点的速度使表达式图像出现

    那就是对于burst height和burst width的调整居然是从两端开始调整的，而不是可以设置为可以从单边调整

使用echospace插件

在repeater中设置旋转、偏移、随机参数

礼花爆炸效果

需要将元素的锚点放在中间，将echospace下的Z轴旋转角度设置好，然后对元素图层的锚点进行距离调整

mirror特效添加之后需要设置镜像中心，直接点击reflection center即可
